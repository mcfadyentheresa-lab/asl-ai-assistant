Create a designer-style template board layout inside my Planning Board for furniture refinishing presentations.

Goal:
Build a reusable, inspirational presentation board that feels curated and high-end — something I can duplicate for different clients, then replace images, colours, notes, and references as needed.

Purpose:
This board is for presenting furniture refinishing ideas to clients. It should feel like a polished concept board, not a technical workspace. It should help me show before/after direction, finish options, colour references, and planning notes in a visually clear way.

-----------------------------------
BOARD STYLE
-----------------------------------

Create a template board layout with a clean, designer-inspired look:
- soft neutral background
- balanced spacing
- layered / collage-style feel
- calm, editorial, high-end presentation
- easy to update and duplicate

The layout should feel inspirational and client-facing, while still being practical for internal planning notes.

-----------------------------------
REQUIRED SECTIONS
-----------------------------------

Build the board with these default zones / areas:

1. TITLE AREA
- space for project or client name
- optional subtitle such as:
  “Furniture Refinishing Direction”
  or
  “Finish Concept Board”

2. BEFORE IMAGE AREA
- clearly labelled “Before”
- image placeholder for the existing piece
- sized large enough to be useful

3. AFTER / DIRECTION AREA
- clearly labelled “After” or “Proposed Direction”
- image placeholder for concept image, inspiration image, or edited preview

4. COLOUR + FINISH SWATCH AREA
Create a structured area for:
- paint colours
- stain colours
- finish samples
- wash directions
- sample labels underneath each swatch

Examples:
- White Wash
- Soft Taupe Wash
- Natural Oak
- Deep Walnut
- BM White Dove
- Custom Blend

5. MATERIAL / TEXTURE REFERENCE AREA
- space for material images or close-up finish references
- examples: wood grain, brushed metal, linen, aged brass, matte finish, etc.

6. PLANNING NOTES AREA
- styled note section for concise planning notes
- should feel clean and intentional, not like a sticky-note mess

Example note types:
- visible grain to remain
- reduce orange/red undertone
- matte protective topcoat
- durability for high-use piece
- sample approval required before final application

7. ARROWS / MARKUP TOOLS
- include arrow-capable annotation support so I can point to details
- arrows should be clean and presentation-friendly
- allow short callouts beside arrows

8. OPTIONAL CLIENT IDEA AREA
- a section where I can drop in client inspiration images / references
- should be visually secondary but still organized

-----------------------------------
BOARD TOOLS / ELEMENTS
-----------------------------------

Use existing Planning Board tools where possible, but configure this template to include or support:

- image placeholders
- text / note cards
- swatch cards
- arrows / connectors
- labels
- layered positioning
- movable items
- easy duplication of cards / sections

If helpful, create reusable card types for:
- Before image card
- After image card
- Swatch card
- Planning note card
- Material reference card

-----------------------------------
LAYOUT REQUIREMENTS
-----------------------------------

The board should open with a default arrangement already in place, not a blank canvas.

Suggested structure:
- top left = title
- left side = before image + client ideas
- centre = after / concept direction
- right side = colour / stain swatches + material references
- bottom or side = planning notes
- arrows and callouts available for markup

Keep it visually balanced and uncluttered.

Do NOT make it feel like:
- a technical admin board
- a generic whiteboard
- a chaotic mood board

It should feel like:
- a curated interior / furniture finish concept board
- something suitable to present to a client
